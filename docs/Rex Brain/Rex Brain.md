# REX VIRAL BRAIN — Sistema Completo de Contenido Viral

Documento unificado para alimentar una AI que genere contenido viral para video corto (TikTok, Reels, Shorts).

---

## 1. Principios de Viralidad

### La Ecuacion

```
VIRALIDAD = ATENCION x RETENCION x EMOCION x IDENTIFICACION
```

Si uno es cero, el video no explota. Los primeros 3 segundos deciden el alcance.

### Por que la gente comparte

| Motivo | Ejemplo |
|---|---|
| Identidad | "Solo los introvertidos entienden esto" |
| Emocion fuerte | Sorpresa, indignacion, humor, inspiracion, FOMO |
| Utilidad | Hacks, trucos, atajos |
| Validacion | "Esto soy yo" |
| Pertenencia | "Si eres freelancer sabras esto" |

### Regla fundamental

**ENTRETENIMIENTO > EDUCACION.** Primero captura atencion. Luego entrega valor.

### Senales del algoritmo

El algoritmo evalua (en orden de peso):
1. Retencion (>70% = empuja a nuevas audiencias)
2. Tiempo de visualizacion
3. Repeticiones
4. Comentarios
5. Compartidos
6. Guardados

---

## 2. Validacion de Idea

Una idea es viable si cumple al menos 6 de 9:

1. Un nino de 10 anos puede entenderla
2. Genera controversia o invita a comentar
3. Esta inspirada en una referencia viral comprobada
4. Alguien pagaria al menos $3 por ver el contenido
5. 50 de 100 personas se identificarian
6. Sin tecnicismos
7. Muestra transformacion clara de A a B
8. Es alcanzable para la persona promedio
9. Si se inspiro en otra idea, tiene un giro propio

---

Todo video tiene UNA intencion principal:

| Intencion | Objetivo | % del calendario |
|---|---|---|
| **VIRALIDAD** | Atraer nueva audiencia | 80% |
| **VALOR** | Educar, generar confianza y autoridad | (incluido en el 80%) |
| **VENTA** | Convertir seguidores en clientes | 20% |

## 6. Sistema de Hooks (Ganchos)

### Los 3 ganchos simultaneos

Todo video viral usa 3 capas de gancho al mismo tiempo:

**1. Gancho Verbal** (<5 segundos)
Formula: `INTERRUPCION + PROMESA + CURIOSIDAD`

**2. Gancho Visual** (1-2 segundos)
Cambio de angulo, movimiento brusco, objeto inesperado, gesto exagerado, transicion rapida, luz/color llamativo.

**3. Gancho Textual** (en pantalla)
Maximo 5 palabras, colores llamativos, sincronizado con lo verbal. Ej: "ERROR", "NO HAGAS ESTO", "SECRETO", "TRUCO".

### Categorias de hooks verbales

**Negativos** — Activan miedo a estar haciendo algo mal:
- No hagas esto en tus videos
- Estas cometiendo este error
- Esto esta arruinando tu contenido
- Si haces esto estas perdiendo dinero
- Este error mata tu alcance
- Deja de hacer esto ahora

**Curiosidad** — Activan necesidad de saber:
- Este truco cambia todo
- Nadie habla de esto
- Este secreto es ilegal / deberia ser ilegal
- Descubri algo increible
- Este hack es absurdo
- Encontre algo oculto

**Contraste** — Activan asombro por transformacion:
- Antes ganaba $0, ahora...
- Pase de esto a esto
- Mira esta diferencia
- Nadie esperaba esto
- Esto parece imposible

**Desafio** — Activan ego y competencia:
- El 99% falla en esto
- Solo el 1% lo sabe
- Apuesto a que no sabias esto
- Nadie puede hacer esto
- Este reto es imposible

**Revelacion** — Activan sensacion de exclusividad:
- El secreto de los creadores grandes
- Lo que nadie te dice
- Esto esta oculto
- La verdad sobre el algoritmo
- Nadie quiere que sepas esto

**Numeros** — Activan estructura y curiosidad:
- 3 errores que cometes
- 5 trucos virales
- 7 hacks de contenido

**Historia** — Activan empatia:
- Casi pierdo todo
- Estuve a punto de rendirme
- Esto cambio mi vida
- Nadie creyo en mi

**Polemicos** — Activan reaccion emocional:
- Instagram esta muriendo
- Los influencers mienten
- La productividad es una mentira
- El algoritmo no es real

### Formula generadora de hooks

Combinar columnas:

| Columna A | Columna B | Columna C |
|---|---|---|
| Este error | arruina | tu alcance |
| Este truco | mejora | tus resultados |
| Este secreto | destruye | tu engagement |
| Este habito | cambia | tu contenido |
| Este mito | explica | tu crecimiento |

Plantillas con variable:
- "No hagas esto si quieres [resultado]"
- "Este error esta arruinando tu [resultado]"
- "El 90% de las personas falla en esto"
- "Nadie habla de esto sobre [tema]"
- "Si haces esto estas perdiendo [beneficio]"
- "Solo el 1% sabe esto"
- "Esto parece falso pero funciona"

## 7. Estructuras de Guion

### 7.1 Estructuras Core (usar como base)

**ERROR FATAL**
Hook → Error comun → Consecuencia → Solucion → CTA

**ANTES VS DESPUES / CONTRASTE**
Hook → Situacion inicial → Cambio clave → Resultado → CTA

**REVELACION**
Hook → Misterio → Contexto → Revelacion → CTA

**LISTA / 3 PASOS**
Hook → Tip/Paso 1 → Tip/Paso 2 → Tip/Paso 3 → CTA

**MITO VS REALIDAD**
Hook → Creencia popular → Desmentir mito → Nueva verdad → CTA

**HACK**
Hook → Problema → Truco → Demostracion → CTA

**HISTORIA CORTA**
Hook → Situacion → Conflicto → Leccion → CTA

**VACIO DE INFORMACION**
Hook intrigante → Pico de curiosidad → Valor inmediato → Respuesta → CTA

**DESAFIO**
Hook → Reto → Intento → Resultado → CTA

**EXPERIMENTO**
Hook → Hipotesis → Prueba → Resultado → CTA

### 7.2 Estructuras Avanzadas

**PROBLEMA INVISIBLE**
Gancho → Problema que el espectador no sabe que tiene → Evidencia → Solucion → CTA

**ESCALERA DE AUTORIDAD**
Gancho antes/despues → Punto inicial → Primer cambio → Estrategia clave → Resultado final → CTA

**MOMENTO WTF**
Gancho impactante → Desarrollo → Giro inesperado → Conclusion → CTA

**DESAFIO CONTRACORRIENTE**
Afirmacion contraria a lo aceptado → Explicacion → Caso real → Nueva perspectiva → CTA

**EFECTO BOOMERANG**
Resultado impactante → Primer error → Segundo error → Revelacion de que fue lo que cambio → CTA

**LOOP DE RETENCION** (mantiene al espectador hasta el final)
Hook → Pregunta abierta → Micro revelacion → Nueva pregunta → Revelacion final → CTA

## 12. Estrategia de Crecimiento

### Reglas operativas
1. Publicar 1-3 videos diarios
2. 80% contenido viral/valor, 20% venta
3. Responder comentarios con video
4. Replicar estructuras de videos virales comprobados con angulo propio
5. Optimizar obsesivamente el primer segundo
6. Crear series de contenido para fidelizar

### Fases de monetizacion

```
CRECIMIENTO (viral) → AUTORIDAD (educativo) → MONETIZACION (venta)
```

### Pipeline de produccion diaria

```
IDEA → HOOK → GUION → VIDEO → PUBLICACION → ANALISIS → OPTIMIZACION → REPETIR
```

### Sistema de escalado
1. Encontrar un video viral de referencia
2. Replicar la estructura
3. Agregar angulo propio
4. Publicar multiples versiones

---

## 13. Checklist Pre-Publicacion

Antes de publicar, el video DEBE cumplir TODOS:

- [ ] El gancho detiene el scroll? (probar con alguien que no lo haya visto)
- [ ] Se entiende el tema en 3 segundos?
- [ ] Genera al menos una emocion fuerte?
- [ ] Hay curiosidad que obliga a seguir viendo?
- [ ] Entrega valor real?
- [ ] Tiene CTA claro?
- [ ] Hay transformacion visible (A → B)?

Si cumple todo → publicar. Si falla en gancho o curiosidad → rehacer antes de publicar.
